# 🚀 Projeto CRUD com Python & Flask

Este é um sistema completo de gerenciamento de itens (Criação, Leitura, Atualização e Deleção) desenvolvido com foco em simplicidade, funcionalidade e um design moderno.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

### Stack Principal
- **Linguagem:** [Python](https://www.python.org/) 🐍
- **Web Framework:** [Flask](https://flask.palletsprojects.com/)
- **Banco de Dados:** SQLite (Armazenamento local leve)
- **ORM:** Flask-SQLAlchemy (para manipulação simplificada de dados)

### Interface (Front-end)
- **HTML5 & CSS3:** Com foco em uma interface premium e responsiva.
- **Jinja2:** Motor de templates para renderização dinâmica no Flask.

## ✨ Funcionalidades

- **Create:** Adição de novos produtos com nome, descrição e preço.
- **Read:** Visualização de todos os itens cadastrados em uma lista moderna.
- **Update:** Edição de informações de itens existentes.
- **Delete:** Remoção de itens do banco de dados.

## 🚀 Como Executar o Projeto

1. Certifique-se de ter o Python instalado.
2. Ative seu ambiente virtual:
   ```powershell
   .\venv\Scripts\activate
   ```
3. Instale as dependências (caso necessário):
   ```powershell
   pip install -r requirements.txt
   ```
4. Execute a aplicação:
   ```powershell
   python app.py
   ```
5. Acesse `http://127.0.0.1:5000` no seu navegador.
